Implementation notes

For the design system v2

Patterns that emerged across these screens — promote them into reusable components in the next design-system iteration.

NEW COMPONENT

TimelineItem

Vertical timeline with date plate + dot states (upcoming · live · past). Used on Events Index (1.2). Should ship in v2 alongside a TodayMarker subcomponent.

NEW COMPONENT

PodiumCard

Gold / silver / bronze top-3 layout. The center card is slightly larger and uses gold-bordered accent. Used on the Leaderboard (3.4).

NEW COMPONENT

ScannerOverlay

QR scanner viewport with corner brackets and animated laser line. Belongs to a dedicated "operator" theme distinct from the public dark palette — slightly deeper black.

NEW COMPONENT

HeroCover

Geometric grid + footprint glyph used as the event cover. No photos. Add to design system with optional date-plate overlay.

NEW COMPONENT

SaveBar (sticky)

"You have unsaved changes" sticky footer with discard / save. Used in Settings (5.4). Add to v2 with a single useSettingsForm hook on the implementation side.

NEW COMPONENT

NotifMatrix

Notification-type × channel grid with switches. Reused in admin event communications. Document the "locked" cell variant.

NEW VARIANT

EventCard · compact

A 56px-date variant for use in 3-column grids and sidebars. Document min/max widths so it doesn't break agenda lists.

NEW PATTERN

Geometric event cover

The teal-gradient + grid overlay + footprint glyph composition. No photos required. Designer-controlled via JSON: {color, glyph, dateOverlay}.

DECISION

Primary token: teal, not indigo

The brief specified indigo but the brand mark is teal — we shifted --primary to oklch(0.70 0.105 192) in dark and oklch(0.58 0.10 192) in light. Brand identity drives UI primary.

GAP — PHASE 2

Speaker CFP application flow

Mentioned in brief as P1.6 but not yet covered. Needs: talk-submission form, status (in review / accepted / declined), and a public CFP page per event.

GAP — PHASE 2

Telegram WebApp views

Native Telegram theming variant. Will require token mapping from CSS variables to Telegram's theme params. Mobile-only.

GAP — PHASE 2

Hackathon platform

Team formation, submission, judging — entire surface area not yet designed. Out of scope for v1 of the portal.
